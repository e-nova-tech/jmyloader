Global loader

A demo of a global loader is visible at the top right of the screen. This loader is set on the body element, and can be used by any element of the page.

Inside a container

This loader is set to be shown inside this container. This is very practical for small portions of page that require a separate loading system.

With Blue theme

This loader is using a different theme : blue. It is easy to implement your own themes from a custom css file.
